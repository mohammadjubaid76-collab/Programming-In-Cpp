# Programming-in-cpp

A collection of foundational C++ programs demonstrating basic syntax, control flow, logical problem-solving, and Object-Oriented Programming (OOP) concepts.

 📁 Repository Structure

The repository contains the following programs, categorized by their core programming concepts:

 1. Basic Logic & Conditionals
* **`if_else.cpp`**: A simple program to check voting eligibility based on age.
* **`odd_even.cpp`**: Checks whether an entered integer is odd or even using the modulus (`%`) operator.
* **`large_two_no.cpp`**: Compares two numbers and identifies the larger one.
* **`Largest_3_Number.cpp`**: Uses nested/else-if logic to find the largest among three numbers.
* **`Income_tax_calculator.cpp`**: A program designed to calculate tax brackets based on annual income (in lakhs).

 2. Loops & Mathematical Logic
* **`Fibonacci_Series.cpp`**: Generates and prints the Fibonacci sequence up to `n` terms using an iterative loop.
* **`Pallindrom_Number.cpp`**: Reverses an integer using mathematical operations to check if it reads the same backward (Palindrome).

 3. Object-Oriented Programming (OOP)
* **`inside_the_class.cpp`**: Demonstrates class structure and how to define a member function outside the class definition using the scope resolution operator (`::`).
* **`Friend_class.cpp`**: Illustrates data encapsulation and how a `friend class` can access the private members of another class.
